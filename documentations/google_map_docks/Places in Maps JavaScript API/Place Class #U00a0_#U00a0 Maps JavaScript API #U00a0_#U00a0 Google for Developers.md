What you can do with the Place class
The Place class has following features:
Search for places
Text Search (New) (/maps/documentation/javascript/place-search) lets you search for place data
using a text query.
Nearby Search (New) (/maps/documentation/javascript/nearby-search) lets you search for nearby
places by place type.
Place Autocomplete (/maps/documentation/javascript/place-autocomplete-overview) provides your
app with the same type-ahead-search behavior of the Google Maps search field.
The Place class provides an API for using the Places library, and supports modern usage patterns
such as Promises (/maps/documentation/javascript/promises). With the Place class, you can create
location-aware features in your app and make detailed location data available to your users
through an intelligent search feature you customize to your needs. The data from the Places
service is one of the most accurate, up-to-date, and comprehensive place models of the real world.

Get place data
Place Details (New) (/maps/documentation/javascript/place-details) provides detailed information
about places, including text summary, operating hours, address, user reviews, photos, and
more.
Place Photos (/maps/documentation/javascript/place-photos) provides access to high-quality
photographic images from the Places database.
Place Reviews (/maps/documentation/javascript/place-reviews) provides access to place reviews
and ratings.
