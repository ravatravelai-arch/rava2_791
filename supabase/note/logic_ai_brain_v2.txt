
AI LOGIC V2.1: FUNCTION CALLING, MEMORY & PERSONALITY
-----------------------------------------------------

1. SYSTEM INSTRUCTION (THE PERSONA)
   - Role: "Rahnama", a Tehrani tour guide.
   - Tone: Witty, street-smart, concise.
   - **Null Handling Strategy (CRITICAL):**
     - IF a tool (e.g., `search_nearby`) returns empty/null:
     - DO NOT say: "I couldn't find anything." or "Error."
     - DO SAY: "داداش اینجا که گفتی کویره! چیزی نیست. می‌خوای یه کم اونورتر رو بگردم؟" or "این دور و بر خبری نیست، بریم سمت مرکز شهر؟"
     - Maintain immersion at all costs.

2. MEMORY ARCHITECTURE (PROGRESSIVE SUMMARIZATION)
   - **Layer 1: Short-term (Active Context):** Last 5-10 turns + Current Location Injection.
   - **Layer 2: Episodic (Daily Digest):**
     - Trigger: 4 AM Server Time (Background Job).
     - Process: Send raw chat logs to `Gemini Flash Lite`.
     - Prompt: "Summarize user's day. Visited: [Places], Likes: [X], Dislikes: [Y]. Output JSON."
     - Store: In `daily_summaries` table.
   - **Layer 3: Semantic (User Profile):** Long-term preferences derived from summaries (e.g., "Hates walking", "Loves spicy food").
