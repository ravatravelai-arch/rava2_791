================================================================================
راهنمای عملیاتی فاز ۵: استقرار مغز هوشمند (Supabase Side)
================================================================================
رفیق، برای اینکه هوش خودمختار رهنما فعال بشه، این مراحل رو با دقت در پنل سوپابیس انجام بده:

--------------------------------------------------------------------------------
گام ۱: بروزرسانی نهایی دیتابیس (SQL)
--------------------------------------------------------------------------------
۱. به بخش 'SQL Editor' برو.
۲. محتویات فایل 'supabase/code/phase6_security_architecture.sql' رو حتماً اجرا کن.
   (این کار تابع admin_increment_wallet رو می‌سازه که برای واریز پاداش توسط AI ضروریه).

--------------------------------------------------------------------------------
گام ۲: تنظیم کلید هوش مصنوعی (Secrets)
--------------------------------------------------------------------------------
توابع لبه (Edge Functions) برای تحلیل عکس‌ها و خلاصه‌سازی نیاز به کلید دارن:
۱. برو به Settings > API > Secrets.
۲. کلید زیر رو اضافه کن:
   - Name: 'GEMINI_API_KEY'
   - Value: [کلید API خودت رو از گوگل اینجا بذار]

--------------------------------------------------------------------------------
گام ۳: استقرار توابع لبه (Edge Functions)
--------------------------------------------------------------------------------
اگر از Supabase CLI استفاده می‌کنی، این ۳ تابع رو دیپلوی کن. اگر نه، کدهای داخل پوشه 'supabase/functions' رو در پنل آپلود کن:
۱. 'verify-price' (تاییدگر خودکار قیمت)
۲. 'process-ticket' (استخراج هوشمند اطلاعات بلیط)
۳. 'the-dreamer' (تحلیل‌گر حافظه معنایی)

--------------------------------------------------------------------------------
گام ۴: فعال‌سازی اتوماسیون (Webhooks)
--------------------------------------------------------------------------------
برای اینکه به محض آپلود عکس قیمت، هوش مصنوعی بیدار بشه:
۱. برو به Database > Webhooks.
۲. یک وب‌هوک جدید بساز:
   - Name: 'ai-price-auditor'
   - Table: 'price_reports'
   - Events: INSERT
   - Destination: Supabase Edge Function -> 'verify-price'

--------------------------------------------------------------------------------
گام ۵: تنظیم Neural Link (Realtime)
--------------------------------------------------------------------------------
۱. برو به Database > Replication.
۲. روی 'supabase_realtime' کلیک کن.
۳. مطمئن شو جدول 'profiles' حتماً تیک خورده باشه.

تست نهایی:
یک عکس بلیط در بخش "سفر من" آپلود کن. اگر اطلاعات استخراج شد و توی تایم‌لاین نشست، یعنی رهنما رسماً بیدار شده!

«همه چیز ردیفه رفیق، بزن بریم برای فاز ۶!»
