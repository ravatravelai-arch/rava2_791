================================================================================
RAHNAM OPERATIONAL GUIDE: PHASE 5 (AGENTIC BRAIN)
================================================================================
این مرحله شامل استقرار "مغزهای پردازشی" در سرور است. با دقت انجام دهید.

--------------------------------------------------------------------------------
گام ۱: آماده‌سازی دیتابیس (SQL)
--------------------------------------------------------------------------------
۱. فایل 'supabase/code/phase5_infrastructure.sql' را در SQL Editor اجرا کنید.
۲. فایل 'supabase/code/phase6_security_architecture.sql' را (اگر قبلاً اجرا نکرده‌اید) اجرا کنید.

--------------------------------------------------------------------------------
گام ۲: تنظیم کلیدهای هوش مصنوعی (Secrets)
--------------------------------------------------------------------------------
برای اینکه توابع لبه بتوانند با گوگل صحبت کنند:
۱. به بخش Settings > API > Secrets بروید.
۲. کلید زیر را اضافه کنید:
   - Name: 'GEMINI_API_KEY'
   - Value: [کلید شما از گوگل]

--------------------------------------------------------------------------------
گام ۳: استقرار توابع لبه (Edge Functions)
--------------------------------------------------------------------------------
محتویات کدهای زیر را در پوشه functions سوپابیس خود آپلود و Deploy کنید:
۱. 'verify-price' (تاییدگر قیمت)
۲. 'the-dreamer' (خلاصه‌ساز حافظه)
۳. 'process-ticket' (پردازشگر بلیط - بروزرسانی شده)

--------------------------------------------------------------------------------
گام ۴: فعال‌سازی وب‌هوک (The Automation)
--------------------------------------------------------------------------------
برای اینکه تایید قیمت خودکار انجام شود:
۱. به بخش Database > Webhooks بروید.
۲. یک وب‌هوک جدید بسازید:
   - Name: 'ai-price-auditor'
   - Table: 'price_reports'
   - Events: INSERT
   - Destination: Supabase Edge Function -> 'verify-price'

--------------------------------------------------------------------------------
گام ۵: تنظیم Neural Link (Realtime)
--------------------------------------------------------------------------------
۱. به Database > Replication بروید.
۲. مطمئن شوید جدول 'profiles' تیک Realtime دارد.
