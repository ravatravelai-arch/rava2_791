RAHNAMA SYSTEM ALIGNMENT GUIDE (POST-PHASE 5 PATCH)
================================================================================

رفیق، جراحی هماهنگ‌ساز با موفقیت انجام شد. برای اینکه این تغییرات توی پنل سوپابیس تو هم اعمال بشه و همه چیز مثل ساعت کار کنه، این ۳ گام ساده رو انجام بده:

۱. اجرای کد امنیتی نهایی (SQL):
فایل 'supabase/code/phase6_security_architecture.sql' رو (که قبلاً برات فرستاده بودم) حتماً مجدداً در SQL Editor اجرا کن. 
این کار تابع 'admin_increment_wallet' رو می‌سازه که Edge Functionهای جدید برای واریز پاداش بهش نیاز دارن.

۲. استقرار فانکشن‌ها (Redeploy):
از اونجایی که من کدهای داخل پوشه 'supabase/functions' رو آپدیت کردم، باید اون‌ها رو دوباره دیپلوی کنی. 
اگر از CLI استفاده می‌کنی:
supabase functions deploy process-ticket
supabase functions deploy verify-price
supabase functions deploy the-dreamer

۳. بررسی Secrets (بسیار مهم):
مطمئن شو که GEMINI_API_KEY حتماً در پنل سوپابیس ست شده باشه:
Settings > API > Secrets

تست نهایی:
یک عکس بلیط تست آپلود کن. اگر توی تایم‌لاین نشست و اروری توی لاگ‌های Edge Function ندیدی، یعنی Neural Link ۱۰۰٪ سینک شده.
